# 视频应用
##  **多字段自适应窗口**
![](./多字段自适应弹窗.jpg)
```javascript
- new Map3dWindow.MultiFieldAdaptWindow(viewer,pisition,title,fields,values)
```
| 参数名称 | 参数类型          | 参数说明       |
| -------- | ----------------- | -------------- |
| viewer   | Cesium.Viewer     | 场景viewer对象 |
| position | Cesium.Cartesian3 | 坐标位置       |
| title    | String            | 窗口标题       |
| fields   | Array.<String>    | 字段列表       |
| values   | Array.<String>    | 值列表         |
